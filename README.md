**使用方法：**

- 在`main.py`中输入您微信小程序登录的账号和密码，一般为**学号和身份证后六位**即可

![账号设置](%E8%B4%A6%E5%8F%B7%E8%AE%BE%E7%BD%AE.png)

- 如果需要邮箱提醒的功能，可以在`mail.py`中输入自己的**邮箱**

  ![邮箱设置](%E9%82%AE%E7%AE%B1%E8%AE%BE%E7%BD%AE.png)

**注意事项**：在使用这个小脚本之前要先取消微信的绑定，否则会报错
